1. Open the mongo database. In my computer, the command is as following: mongod.exe --dbpath=D:\mongodb\data\db
2. Enter the final file. Install the node modules using the following command: npm install
3. Seed the data. (Adding products information into database) Using the command: node seed.js
4. Run the server using command: npm start
5. Open the browser, enter the url: http://localhost:3000
Now you can use the website. On the website, you can register and log in to buy something, reduce or add something in shoppig cart, finally you can check out to pay the items.
